

# Table Relay

A multi-database desktop workbench — SQL editor, schema diagrams, and an AI assistant across MySQL, Postgres, SQLite, MongoDB, and Redis. Built with Tauri.

## Run Locally

**Prerequisites:** Node.js, Rust (for Tauri)

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run tauri:dev`
