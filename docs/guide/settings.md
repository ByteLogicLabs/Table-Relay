# Settings

Open settings from the native menu **Settings…** (`Cmd/Ctrl+,`) or the home
screen. The dialog has six sections.

## Appearance

- **Theme** - seven built-in themes: One Dark, Catppuccin Latte, Monokai,
  Dracula, Nord, Tokyo Night, GitHub Dark. (The data grid and Monaco editor
  retheme to match.)
- **Sidebar** - the connection rail mode: **Auto** (hover-expand), **Expanded**,
  or **Collapsed**.

## General

- **Restore on startup** - reconnect the last-focused connection when the app
  launches.
- **Default row limit** - 50 / 100 / 250 / 500 / 1000.
- **NULL display** - how `NULL` shows in the grid (blank, the text `NULL`, or a
  symbol).

## Editor

Monaco preferences: font size, tab indent width (2 / 4 / 8), word wrap, minimap,
and autocomplete.

## AI Providers

Manage AI credentials and behavior - see [AI assistant](ai-assistant.md):

- Add / edit / delete provider credentials (OpenAI, Anthropic, Gemini,
  OpenAI-compatible, Local Llama, and CLI providers) and set the active one.
- Default **effort** level for the assistant.
- **AI permissions** persistence (whether auto-approval choices are remembered
  across restarts) - see [AI safety](ai-safety.md).

## Import / Export

Export and import your **connections** (optionally password-encrypted), and
transfer them between machines. See [Import & export](import-export.md).

## Logs

View and copy the app log and the chat log, and open the log file locations.
