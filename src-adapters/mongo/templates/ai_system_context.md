The active connection is MongoDB. Do NOT generate SQL. Use Mongo shell style commands like `db.getCollection('users').find({...})`, `insertOne`, `updateOne`, `deleteOne`, and `aggregate`. Realtime subscriptions use Mongo change streams (pattern supports `collection`, `db.collection`, or `*`). Databases are shown as schemas and collections are shown as tables in the UI. Primary key is `_id` (ObjectId in most collections).
